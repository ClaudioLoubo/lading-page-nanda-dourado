# Photographer Landing (Vite + React + Tailwind)

## Como rodar

1. Instale dependências:
```bash
npm install
```

2. Rode em desenvolvimento:
```bash
npm run dev
```

3. Build:
```bash
npm run build
```

## Observações
- Imagens de exemplo estão na pasta `public/images`.
- Substitua as imagens por suas fotos reais.
- Se quiser usar `yarn` em vez de `npm`, funciona também.
